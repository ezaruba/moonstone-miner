To use our miner, you need to configure it in accordance with our recommendations:

1. Download Moonstone Miner;
2. Select “Custom”  in the main menu;
3. In the Custom miner configuration, please keep the recommended settings;
4. In the Install Url field enter
https://raw.githubusercontent.com/valar999/moonstone-miner/master/moonstone-1.2.30.tgz
5. In the Algo field enter ethash
6. In the Template field enter % WAL%.% WORKER_NAME%
7. In the Url: pool.minerall.io field enter 3401
8. In the User Config field enter --cuda-continuous-mode

Now you can fully appreciate effectiveness of Moonstone Miner!
